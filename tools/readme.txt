เครื่องมือที่ใช้ในการพัฒนา
1.Eclipse เป็น IDE สำหรับพัฒนาโปรแกรม 
2.Android Software Development Kit (Android SDK)
ชุดคำสั่งในการพัฒนา Applications บนระบบปฏิบัติการ Android
3.Android Development Tools Plug-in (ADT)  
ชุดคำสั่งที่รวมเอาโปรแกรมช่วยพัฒนารวมถึงตัวจัดการสิ่งแวดล้อม
สำหรับการพัฒนา ทำให้สามารถสร้างและDebug Android Applications

แหล่งที่มาของ Source Code อื่นๆที่มาประกอบในโปรแกรม
บน Android
1.ZXing BarcodeScanner  สำหรับการอ่าน QR-Code
2.Ksoap2-android สำหรับการเชื่อมต่อกับ Facebook Web Application
3.Facebook Android SDK สำหรับการติดต่อกับ Facebook Account

บน Facebook Web Application 
1.NuSOAP - SOAP Toolkit for PHP สำหรับการเชื่อมต่อกับ Android
http://sourceforge.net/projects/nusoap/
2.Google Chart API สำหรับสร้าง QR-Code
http://code.google.com/apis/chart/
3.Facebook PHP SDK สำหรับการติดต่อกับ Facebook Account
https://github.com/facebook/php-sdk/
